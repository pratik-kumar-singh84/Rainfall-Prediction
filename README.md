# Rainfall-Prediction
This project predicts whether rainfall will occur based on historical weather data using Machine Learning algorithms. The model analyzes different weather parameters such as temperature, humidity, pressure, wind speed, and other atmospheric conditions to make accurate rainfall predictions. The goal is to assist in weather forecasting and support decision-making in agriculture, disaster management, and water resource planning.

# Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Rainfall Prediction
- Data Visualization
- User-friendly Jupyter Notebook

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

# Dataset

The project uses historical weather data containing features such as:

- Temperature
- Humidity
- Pressure
- Wind Speed
